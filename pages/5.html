B. OS Structure

So far, we have discussed the reason why Operating Systems were made, but, we have only covered 2 parts: System Libraries and Device Drivers. An OS is made of these parts:

  • System Libraries: The way for developers to use the OS's functionality

  • Device Drivers: How an OS talks to real hardware

  • Software: This includes basic apps eg. calculator, notepad, etc. and system apps, like your graphical desktop environment, mouse renderer, audio servers, display servers, etc.

  • Kernel: The heart of the OS. The actual thing that handles Library calls, interacts with drivers and runs the OS.

With all these main components and many more, the OS becomes fully functional and usable
